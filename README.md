# vue食堂移动端项目：

### 背景：

- 疫情期间尽量避免人群和人群密集的场合。因而我们食堂需要一种途径可以减少人流量。
- 学生在食堂就餐时，可能会集中在一个时间段内，会造成波峰式的排队、就餐。



### 目标：

- 食堂提前预览菜单、在手机上下订单，饭好后到食堂领取，回寝室就餐。
- 学生可以选择食堂、餐品、以及取餐时间。食堂人员可以看到订单、进行准备。
- 通过这个食堂项目，实现将食堂人数由波峰式转为分批次就餐。
